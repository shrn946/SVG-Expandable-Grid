SVG Expandable Grid is a WordPress plugin designed to enhance your website by incorporating an engaging expandable post grid feature. With this plugin, you can effortlessly showcase your posts in a visually appealing grid layout, allowing users to click and explore each post with a captivating expandable effect.

Features
User-Friendly Shortcode: Easily integrate the expandable grid into your posts or pages using the [wp_expandable_grid] shortcode.

Customization Options: Tailor the grid to your preferences by utilizing shortcode attributes, such as specifying a post category, controlling the number of posts displayed, and sorting criteria.

Shortcode Examples
Basic Usage:

csharp
Copy code
[wp_expandable_grid]
Filtered by Category and Limited Posts:


Copy code
[wp_expandable_grid category="podcast" posts_per_page="5"]
Installation
Upload the svg-expandable-grid folder to the /wp-content/plugins/ directory.
Activate the plugin through the 'Plugins' menu in WordPress.
